Este repositorio contiene el desarrollo de un Sistema de Gestión para un Centro Integral, realizado como proyecto de simulación profesional (Visita Empresa Rooftop) por los alumnos de 6° Año "B" en la materia Programación III.

El proyecto simula el flujo de trabajo de una empresa real de software, aplicando metodologías ágiles y estándares modernos de desarrollo.

Contexto Educativo
Institución: Colegio PROA

Materia: Programación III

Curso: 6° Año División "B"

Objetivo: Integrar conocimientos de lógica de programación, estados globales y componentes reutilizables en un entorno de simulación laboral.

Tecnologías Utilizadas
Framework: Vue.js (v3)

Arquitectura: Component-Based Architecture.

Gestión de Estado: [Pinia / Vuex]

Estilos: [CSS3 / Tailwind / Bootstrap]

Herramientas de Desarrollo: Vite / npm.

Funcionalidades del Sistema
Gestión de Turnos: Interfaz dinámica para la reserva y visualización de citas médicas/administrativas.

Panel de Administración: Control de usuarios, servicios y especialistas del centro.

Validación de Datos: Formularios reactivos para el ingreso de información de pacientes.

Interfaz Adaptable: Diseño pensado para ser utilizado en diferentes dispositivos (Desktop/Tablet).

Instalación y Configuración
Para ejecutar este proyecto localmente, seguí estos pasos:

Clonar el repositorio:

Bash
git clone https://github.com/Ortiz21402098/CentroIntegral.git
Instalar dependencias:

Bash
npm install
Iniciar el servidor de desarrollo:

Bash
npm run dev
